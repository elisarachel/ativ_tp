## Instruções de execução
### 1. Clone o repositório
```
git clone https://github.com/elisarachel/atv_tp.git
```
### 2. Entre na branch da atividade I
```
git checkout atvi
```
### 3. Instale as dependências
```
npm install
```
### 4. Entre na pasta out
```
cd out
```
### 5. Rode o arquivo ```main.js```
```
node main.js
```
